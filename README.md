# Object-Oriented Programming Labs

This is a compilation of the labs completed for the Object-Oriented Programming Course (Fall 2025).

## Lab 2 – User Stories
**Objective:** Develop user stories for a ride-sharing application.  
**Summary:** Created detailed user stories that define the key actions and goals of users within the ride-share system.

## Lab 3 – Use Case Diagrams
**Objective:** Visualize system functionality using UML.  
**Summary:** Designed a use case diagram for the ride-sharing application.

## Lab 4 – Class Diagrams
**Objective:** Translate system requirements into structured classes.  
**Summary:** Created a class diagram for the ride-sharing app based on pre-determined class structures. The diagram included attributes, methods, and relationships.

## Lab 5 – Working with Classes and Data in Python
**Objective:** Implement classes and manage data.  
**Summary:** Developed Python classes to organize and manage data stored in JSON format.
